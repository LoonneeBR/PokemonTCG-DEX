# PokemonTCG-DEX

Track your Pokémon TCG collections in a simple and visual way.

# Features
📖 Multiple expansions
🎴 Card gallery mode
📋 List mode
⭐ Foil card tracking
🔴 Owned card tracking
📊 Collection progress statistics
💾 Automatic save with LocalStorage
📱 Responsive design
# Built With
HTML5
CSS3
JavaScript (Vanilla)
